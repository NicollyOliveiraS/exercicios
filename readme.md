## Descrição
Este projeto reúne três pequenas aplicações web desenvolvidas para auxiliar em cálculos corporativos comuns:
- Cálculo de Horas Extras

- Cálculo de Aumento Salarial

- Cálculo de Transporte Fretado

Cada aplicação é acessível por meio de uma página inicial  que lista os três projetos.
O objetivo é oferecer ferramentas simples e funcionais para gestores e profissionais de RH, facilitando a tomada de decisão com base em regras de negócio pré-definidas.


## Funcionalidades
1. Cálculo de Horas Extras
Informar salário mensal do funcionário.

Informar quantidade de horas extras em dias úteis.

Informar quantidade de horas extras em finais de semana.

Cálculo automático do valor total (salário + horas extras), considerando:

Hora normal = salário / 200

Hora extra dia útil = valor hora × 1,5

Hora extra fim de semana = valor hora × 2

2. Cálculo de Aumento Salarial
Informar salário atual do funcionário.

Aplicar aumento conforme faixas:

Até R$ 1.200,00 → 16%

R$ 1.200,01 a R$ 2.100,00 → 13%

R$ 2.100,01 a R$ 3.000,00 → 10%

Acima de R$ 3.000,00 → 5%

Exibir o novo salário após o aumento.

3. Cálculo de Transporte Fretado
Informar quantidade de colaboradores.

Informar quantidade de dias úteis.

Calcular valor total do transporte com base na faixa de quantidade:

1 a 49 pessoas → R$ 4,50 por pessoa/dia

50 a 99 → R$ 4,10

100 a 149 → R$ 3,80

150 ou mais → R$ 3,60
